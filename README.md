# starter-react-mobx-css-modules
My starter kit for projects based on React, MobX, CSS Modules and TypeScript.

This kit consists of the latest versions of:
* React
* Webpack
* MobX
* SCSS
* CSS Modules
* TypeScript

#### Build Instructions

`yarn install`

`yarn run build`

The output goes to `/build/bundle.js` & `/build/index.html`

#### Development Server

`yarn run dev-server`
